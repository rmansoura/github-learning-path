## üìñ Books & Articles

  https://git-scm.com/book/en/v2
  - Comprehensive reference
  - Covers Git internals, workflows, and best practices

- **Conventional Commits**:
  https://www.conventionalcommits.org/
  - Guide to semantic commit messages
  - Improves readability and enables automation

- **Git Flow** (Original Article):
  https://nvie.com/posts/a-successful-git-branching-model/
  - The Git Flow branching strategy explained
  - Best for planned releases

- **GitHub Flow** (Guide):
  https://guides.github.com/introduction/flow/
  - Simpler alternative to Git Flow
  - Recommended for continuous deployment

- **Semantic Versioning**:
  https://semver.org/
  - Version numbering best practices
  - MAJOR.MINOR.PATCH scheme
